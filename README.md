# SafeCab - Women Safety Cab Booking System

SafeCab is a responsive Women Safety Cab Booking System website built using HTML, CSS, and JavaScript. It focuses on safer ride booking experiences with live tracking, SOS emergency support, trusted contact sharing, AI chatbot assistance, voice booking, and secure payment options.

## Features

- Responsive landing and splash page
- SafeCab login and signup pages
- Dynamic user greeting after login/signup
- Ride booking with pickup and drop location
- Dynamic fare, distance, and travel time estimate
- Google Maps route preview
- Live tracking page with moving cab animation
- Trusted contacts ride sharing
- Browser notification and live tracking link sharing
- SOS emergency page with alert UI
- Safety recording page
- AI driver safety score page
- Driver Trust Card with verified badge, rating, and experience
- Route deviation alert popup
- AI chatbot with logged-in user greeting
- Voice booking assistant
- Women driver preference page
- Payment page with UPI QR, card, wallet, and cash options
- Logout option from user menu

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Google Maps Embed
